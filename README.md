# Enjoybacon7 Tap

## How do I install these formulae?

`brew install enjoybacon7/tap/<formula>`

Or `brew tap enjoybacon7/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "enjoybacon7/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
